# التلقين العربي v0.1.0 — الإصدار الأول

أول إصدار عام من **التلقين العربي** — تطبيق تلقين احترافي مصمم من الألف إلى الياء للمحتوى العربي.

## لماذا هذا التطبيق؟

تطبيقات التلقين الموجودة في السوق لا تدعم اللغة العربية بشكل صحيح — اتجاه النص معكوس، التشكيل يختفي، والخطوط غير مناسبة. **التلقين العربي** يحل هذه المشاكل بتطبيق مبني خصيصاً للعربية كلغة أولى.

## المميزات الرئيسية

### ✏️ محرر نصوص غني
- تنسيق كامل: خط عريض، مائل، تحته خط
- ألوان النص والتظليل
- فواصل المشاهد والمقاطع
- استيراد من ملفات TXT و DOCX (سحب وإفلات)

### 🔤 دعم عربي متكامل
- **التشكيل الكامل** — اعرض أو أخفِ الحركات بضغطة واحدة
- **اتجاه ثنائي** — كل فقرة باتجاهها الصحيح (RTL للعربية، LTR للإنجليزية)
- **خطوط عربية احترافية** — أميري، القاهرة، نسخ عربي، تجوال
- **وضع المرآة** — لشاشات التلقين الزجاجية

### 🖥️ شاشة مزدوجة
- نافذة منفصلة للمقدم (شاشة العرض)
- تزامن فوري بين النافذتين
- شاشة العرض بدون عناصر تحكم — نص فقط

### ⚙️ تخصيص كامل
- حجم الخط (٢٤ – ١٢٠)
- سرعة التمرير
- ارتفاع السطر والهوامش
- خط الإرشاد الأحمر للقراءة

### 🛡️ خصوصية وأمان
- يعمل **بدون إنترنت** بالكامل
- البيانات تبقى على جهازك فقط
- استعادة تلقائية للجلسة عند إغلاق غير متوقع

## الأنظمة المدعومة

| النظام | المعالج | الصيغة |
|--------|---------|--------|
| macOS | Apple Silicon (M1+) | `.dmg` |
| macOS | Intel (x86_64) | `.dmg` |
| Windows | x64 | `.msi` / `.exe` |
| Linux | x64 | `.AppImage` / `.deb` |

### متطلبات النظام
- **macOS:** 10.15 (Catalina) أو أحدث
- **Windows:** Windows 10 أو أحدث
- **Linux:** Ubuntu 20.04 أو أحدث (أو ما يعادلها)
